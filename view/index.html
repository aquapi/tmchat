TMChat

Provides temporary chat room API.

This project is built based on BunSVR.

Message

Message format: [action].[room].[author].[message]

Actions
  • Join room: j
  • Leave room: l
  • Send message: t
  • Ping: p
Examples
  • Join room general: j.general
  • Leave room general: l.general
  • Send Hello world! in room general with username aquapi: t.general.aquapi.Hello world!
Response
  • Invalid message: M
  • Invalid author: A
  • Invalid room: I
  • Message: [author]:[message]
