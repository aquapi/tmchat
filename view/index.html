TMChat

Provides temporary chat room API.

This project is built based on BunSVR.

Message

Message format: [action].[room].[message]

Actions
  • Join room: j
  • Leave room: l
  • Send message: t
Examples
  • Join room general: j.general
  • Leave room general: l.general
  • Send Hello world! in room general: t.general.Hello world!
